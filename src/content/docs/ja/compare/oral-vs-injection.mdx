経口エストラジオール（プロギノバ／吉草酸エストラジオール錠）と吉草酸エストラジオール筋注（EV）は、中国語圏MTFで最も一般的なE2の2経路です。本ページでは薬物動態、安全性、実践的ハードルを比較し、**普遍的な推奨は提示しません**。

:::note[30秒判断マトリックス]
| 該当する状況は → | 推奨選択 |
|---|---|
| VTEリスクを最低にしたい／年齢≥40／喫煙／BMI>30 | **注射EV** · 肝初回通過を回避、RR ≈ 1 |
| 注射が怖い／生活リズムが不規則で打つ日を覚えにくい／手技を学びたくない | **経口プロギノバ** · ハードルゼロ |
| VTE既往／前兆を伴う片頭痛／第V因子ライデン | **注射EV**（または経皮） · 経口は禁忌 |
| 中国本土で入手しやすい＋手間をかけたくない | **経口プロギノバ** · 婦人科で常備 |
| 単剤治療を目指す（抗アンドロゲンなし） | **注射EV** · 約82%がT&lt;50 ng/dLに到達 |
| 注射間隔が不規則／日内をより安定させたい | **経口BID** · 注射はピーク／トラフ比3-4:1 |
| 妊娠／授乳／18歳未満 | いずれも非推奨。受診してください |

> 迷う場合は詳細比較を続けて読むか、[リスク自己評価](/ja/tools/risk-screener/)を参考にしてください。
:::

## ひと目でわかる

| 項目 | 経口（プロギノバ） | 注射（吉草酸エストラジオールEV） |
|---|---|---|
| **肝初回通過代謝** | あり（通過） | なし（回避） |
| **VTEリスク** | ベースラインの2-4倍 | おおむねベースライン |
| **投与頻度** | 1日1-3回 | 5-10日ごとに1回 |
| **血中濃度の安定性** | 1日内にピーク／トラフあり | 投与間隔内で大きな変動 |
| **効果発現** | 数時間でピーク | 24-48時間でピーク |
| **エビデンスレベル** | <EvidenceBadge level="A" /> | <EvidenceBadge level="A" /> |
| **中国本土での入手** | 処方薬、婦人科／内分泌科で常備 | 処方薬、一部の病院では処方を渋る |
| **月額費用** | 約20-60元 | 約30-100元 |
| **手技ハードル** | ゼロ | 自己筋注の習得が必要 |

## 薬物動態

### 経口：肝初回通過が曲線を形作る

経口エストラジオールまたは吉草酸エストラジオールは小腸で吸収され、門脈を経て肝臓に入り、**初回通過代謝で50-70%が失われます**<CitationRef id="kuhl-2005" />。体循環に到達する有効用量は服用量の30-50%にすぎず、肝臓は高濃度のE2/E1に「浸される」ことになります。

- 経口2-4 mgで血中濃度ピークは約50-150 pg/mL、4-8時間後に低下
- 半減期は約12-14時間（E2として）、ただしE1/E2比が高くなる
- 肝臓でSHBG、凝固因子、CRPの産生が増加 → VTE／胆汁うっ滞リスク上昇<CitationRef id="canonico-2018" />

### 注射：筋肉内での徐放、より平坦な曲線

吉草酸エストラジオール筋注後、油性溶剤が筋肉内にデポを形成し、エステル結合がゆっくり加水分解されてE2が放出されます<CitationRef id="aly-2021" />：

- 4 mg EV筋注で24-48時間後に約300-600 pg/mLにピーク、その後指数関数的に減衰
- 半減期は約4-5日；5-7日後に100-150 pg/mLまで低下
- **肝初回通過を完全に回避**、凝固への影響は生理的エストラジオールに近い

## VTEリスク比較

大規模MTFコホートと閉経後HRT研究に基づく<CitationRef id="canonico-2018" /><CitationRef id="coleman-2022" />：

| 経路 | VTE相対リスク（ベースライン比） |
|---|---|
| 経口エストラジオール | **2-4倍** |
| 結合型エストロゲン（CEE、廃れた製剤） | 4-6倍 |
| 舌下投与 | 1.5-3倍の可能性（エビデンス限定的） |
| 経皮ジェル／パッチ | **1倍（ベースライン）** |
| 筋注吉草酸エストラジオール | **1倍（ベースライン）** |

**絶対リスクは依然低い**：若年で健康なMTFのベースラインVTEは約1-2/1000人年。喫煙＋肥満＋40歳超が重なると絶対リスクは大幅に上昇します。

**ガイドラインの立場**<CitationRef id="coleman-2022" /><CitationRef id="hembree-2017" />：
- 年齢≥40歳＋いずれかのリスク要因 → 非経口を優先
- VTE既往 → 経口を回避、経皮／注射＋抗凝固評価を優先
- 術前2-4週 → 経皮へ切替または休薬

| 経路 | 日内変動 | 日間変動 | 症状への影響 |
|---|---|---|---|
| 経口1日1回 | ピーク／トラフ比 ~3:1 | 小 | 気分の波・周期的なむくみ感 |
| 経口1日2回 | ピーク／トラフ比 ~2:1 | 小 | より平坦 |
| 注射7日ごと | なし | ピーク／トラフ比 ~3-4:1 | 「打った後数日のハイ」 |
| 注射5日ごと | なし | ピーク／トラフ比 ~2:1 | 安定状態が良好 |
| パッチ／ジェル | 小 | 小 | 最も安定 |

**選択ロジック**：気分や体感の安定を求めるユーザーはパッチ／ジェル／短間隔注射を好み、用量の柔軟性に敏感なユーザーは経口を好みます。

## 効果発現と切替

### 経口の効果発現

- 服用後2-4時間でピーク、乳房発達や皮膚変化などの組織反応は1-3ヶ月で出現開始
- 安定状態のT抑制は約4-8週（抗アンドロゲン併用時）

### 注射の効果発現

- 初回注射は24-48時間でピークだが、単回では安定状態に到達しない
- 2-3回投与後（約10-20日）に安定状態に近づく
- 切替の移行期はオーバーラップを推奨：初回注射の1-2週前から経口を維持し、注射後に経口を漸減

### 相互の切替

**経口 → 注射**：
1. 等価用量を計算：経口2-4 mg/日 ≈ EV 4-8 mgを7日ごと
2. 初回注射後2-4日は経口を維持し、その後漸減
3. 注射後4-6週でトラフのE2を測定し、安定状態を確認

**注射 → 経口**：
1. 最後の注射を中止し、5-7日待つ（血中濃度低下）
2. 経口を開始、初回用量は安定状態の注射等価量の1-1.5倍が一般的（肝初回通過を補正）
3. 2-4週で血液検査して調整

## 実践的ハードル

### 経口

- ハードルはゼロ、時間どおりに服用するだけ
- 注：プロギノバは吉草酸エストラジオール2 mg/錠であり、「エストラジオール2 mg」と等価、換算不要
- 飲み忘れ：12時間以内なら追加服用、12時間を超えたらスキップ

### 自己注射

- 油性製剤の筋注：23G 1-1.5インチ針（約25-38 mm）、薬液吸引は21G
- 注射部位：殿部背外側（鼠径部後上外象限）、大腿前外側、三角筋（少量時）
- 消毒：70%アルコール、使用前にバイアル栓を30秒拭き、注射部位は時計回りに拭く
- コツ：気泡を抜く、ゆっくり注入（10-30秒）、抜針後30秒押さえる

<WarningBox>
**初回注射は強く推奨**：友人を同席させる・救急電話を準備・30分間アレルギーや血管迷走神経反応の有無を観察。詳細な手順は当サイトの[初回注射ガイド](/ja/guides/first-injection/)を参照。
</WarningBox>

### 皮下注射（SC）の代替

油性吉草酸エストラジオールは**皮下注射**（腹部、大腿前面）が可能で、痛みが軽く半減期がやや長い。針は25G × 5/8インチ（16 mm）。エビデンスレベルは中等度で、一部のガイドラインは受け入れている<CitationRef id="aly-2021" />。

## 中国本土での入手

| 経路 | 経口（プロギノバ） | 注射（EV） |
|---|---|---|
| 婦人科 | 常備、処方しやすい | 一部の病院では処方しない |
| 内分泌科 | 比較的容易 | 医師の姿勢次第 |
| 生殖センター | 非常に容易 | 排卵誘発スキームでよく使われる |
| 精神科（トランス外来） | 処方可 | 処方可 |
| 費用 | 20 mg × 30錠で約40元 | 5 mg/mL × 10アンプルで約60-100元 |
| 保険適用 | 部分償還 | 部分償還 |

## 選択方法（臨床判断フレームワーク）

**経口を優先する状況**：
- 若年、VTEリスク要因なし、筋注を学びたくない
- 注射の入手が困難または医師が処方を渋る
- 短期の橋渡し、数ヶ月後に切替予定
- 「毎日服薬」という形式に抵抗がない

**注射を優先する状況**：
- 年齢≥40またはVTEリスク要因あり（喫煙／肥満／家族歴）
- 経口吸収が不安定（消化管手術歴、IBD）
- より平坦な血中濃度を求める
- 自己注射の習得に意欲的

**パッチ／ジェルを優先する状況**（本ページで未比較だが、しばしば最適）：
- VTE既往または高リスク
- 針を完全に避けたい
- 用量微調整に敏感
