CPA（酢酸シプロテロン、通称アンドロクール）とスピロノラクトン（アルダクトン）は、中国語圏のMTF HRTで最も使用されている2つの抗アンドロゲン薬です。本ページでは臨床判断軸ごとに項目別に比較します。**普遍的な推奨は提示しません** — 選択は医師が個別の併存疾患、モニタリング条件、薬剤入手性に基づいて判断すべきです。

:::note[30秒判断マトリックス]
| 該当する状況は → | 推奨選択 |
|---|---|
| 血清Tを女性範囲まで素早く下げたい | **CPA 5-12.5 mg** · 負のフィードバック抑制がより確実 |
| 抑うつ傾向／肝機能異常／長期使用（5年超）を計画 | **スピロノラクトン** · 髄膜腫の累積リスクなし |
| ACEI/ARB併用または高カリウム食 | **CPA** · スピロノラクトンのカリウム保持作用が高K血症リスクを増悪 |
| 腎機能不全（eGFR &lt;60） | **CPA**（減量） · スピロノラクトンは禁忌 |
| 中国本土で入手しやすい／経済的考慮 | **スピロノラクトン** · 国産後発品が広く流通、保険適用 |
| 髄膜腫の既往／ベースラインMRI異常 | **スピロノラクトン** · CPAは絶対禁忌 |
| 妊娠／授乳／18歳未満 | いずれも非推奨。受診してください |

> 迷う場合は詳細比較を続けて読むか、[リスク自己評価](/ja/tools/risk-screener/)を参考にしてください。
:::

## ひと目でわかる

| 項目 | CPA（アンドロクール） | スピロノラクトン（アルダクトン） |
|---|---|---|
| **作用機序** | プロゲステロン受容体作動＋弱いAR拮抗＋LH/FSH抑制 | アルドステロン拮抗（カリウム保持性利尿）＋高用量AR拮抗 |
| **T抑制効果** | より確実（下垂体への負のフィードバック） | 中等度（受容体ブロック中心、血清Tは去勢域に達しないことも） |
| **標準用量** | 5-12.5 mg/日 | 100-300 mg/日 |
| **エビデンスレベル** | <EvidenceBadge level="A" /> | <EvidenceBadge level="A" /> |
| **最も重大な長期リスク** | 髄膜腫（累積用量関連） | 高カリウム血症（腎機能関連） |
| **モニタリング重点** | 肝機能・プロラクチン・累積用量・頭痛/視覚症状 | 血清カリウム・腎機能・血圧 |
| **中国本土での入手** | 処方薬、精神科／内分泌科で処方可 | 循環器内科／腎臓内科で常備、比較的入手容易 |
| **月額費用** | 約30-80元 | 約10-30元 |

## 作用機序の違い

### CPA：多重抑制、負のフィードバック中心

CPAは3つの機序でアンドロゲン作用を低減します<CitationRef id="meyer-2020" />：

1. **プロゲステロン負のフィードバック**（主要）：視床下部GnRH抑制→下垂体LH/FSH低下→テストステロン合成減少
2. **弱いアンドロゲン受容体拮抗**：効力はビカルタミドよりはるかに弱く、臨床的意義は限定的
3. **軽度の5α還元酵素阻害**：T→DHT変換を減少

結果：血清テストステロンは通常女性範囲（&lt;50 ng/dL）まで低下し、エストロゲンとの併用でより安定します。

### スピロノラクトン：アルドステロン＋アンドロゲンの二経路

スピロノラクトンはもとはカリウム保持性利尿薬であり、その抗アンドロゲン作用は高用量でのアンドロゲン受容体への競合的阻害から生じます<CitationRef id="hembree-2017" /><CitationRef id="ucsf-2016" />：

1. **アルドステロン受容体拮抗**（主機序）：カリウム保持性利尿、降圧
2. **アンドロゲン受容体拮抗**（高用量）：100 mg以上で出現
3. **テストステロン合成の軽度抑制**：機序は完全には解明されていない

結果：血清テストステロンは中等度の低下にとどまるか正常範囲を維持することもありますが、組織レベルでアンドロゲン作用がブロックされ、臨床的女性化効果は得られます。

## T抑制効果の比較

| 指標 | CPA 5-12.5 mg | スピロノラクトン 100-300 mg |
|---|---|---|
| 血清テストステロン中央値の低下 | 80-95% | 30-60% |
| 女性範囲（&lt;50 ng/dL）到達割合 | 高い | 低い、E2併用が必要なことが多い |
| 効果発現時間 | 4-8週 | 6-12週 |
| DHTへの影響 | 中等度 | AR阻害により間接的に低下 |

**ポイント**：「血清数値志向」のユーザー（T値を女性範囲まで下げたい）にはCPAの方が達成しやすく、「症状志向」のユーザー（体毛、皮脂、性欲を重視）には両者とも有効です。

## 副作用プロフィール

### CPAの主要リスク

<DangerBox title="髄膜腫 — 累積用量関連">
EMA 2020年の使用制限<CitationRef id="ema-2020" />は、フランス全国コホート（Meyer 2020）<CitationRef id="meyer-2020" />と後続研究<CitationRef id="lee-2022" />に基づきます：

- **≥25 mg/日を1年以上累積**：OR = 12.4（95% CI 5.9-26.1）
- **累積用量≥60 g**：リスクが有意に増加
- トランス領域で常用される5-12.5 mg/日では累積用量は低めですが、長期使用ではモニタリングが必要

**警告症状**：持続性頭痛、視野欠損、痙攣、人格変化 → ただちに受診し頭部MRIを実施。
</DangerBox>

その他のリスク：
- **肝毒性**：ALT/AST上昇、まれに急性肝不全；3-6ヶ月ごとに肝機能検査
- **抑うつ／気分低下**：プロゲステロン作用、約10-20%のユーザーが報告
- **プロラクチン上昇**：PRLをモニタリング、持続上昇は減量を示唆
- **性欲低下**：T著明低下による予測される効果、一部のユーザーには困りごと

### スピロノラクトンの主要リスク

<WarningBox title="高カリウム血症 — 腎機能がリスクを決定">
スピロノラクトンはアルドステロンを抑制 → 腎臓でカリウム保持 → 血清カリウム上昇。リスク要因：

- 腎機能低下（eGFR &lt;60）
- ACEI/ARB、NSAIDs、カリウム製剤の併用
- 高カリウム食品の過剰摂取（ココナッツウォーター、バナナ、減塩塩）
- 脱水、激しい運動後

**モニタリング**：開始時2週、用量変更後2週、安定後3-6ヶ月ごとに血清カリウム＋クレアチニン。
</WarningBox>

その他のリスク：
- **多尿／夜間頻尿**：利尿作用、多くは4-8週で慣れる
- **低血圧／起立性めまい**：開始期に顕著
- **乳房痛**：よくみられ、通常は中止不要
- **性欲の変動**：個人差が大きい

## 禁忌

| 禁忌 | CPA | スピロノラクトン |
|---|---|---|
| 髄膜腫の既往 | 絶対禁忌 | 安全 |
| 重度の肝機能不全 | 禁忌 | 慎重投与 |
| 腎機能不全（eGFR &lt;30） | 減量 | 禁忌 |
| 既知の高カリウム血症 | 安全 | 禁忌 |
| ACEI/ARB併用でモニタリング不可 | 安全 | 禁忌 |
| うつ病／双極性障害の活動期 | 慎重投与 | 比較的安全 |
| 妊娠（妊孕性のあるユーザー） | 禁忌 | 禁忌 |

## モニタリングスケジュール

| 時点 | CPA | スピロノラクトン |
|---|---|---|
| ベースライン | 肝機能・プロラクチン・MRI（任意） | 血清カリウム・腎機能・血圧 |
| 第2週 | — | 血清カリウム・血圧 |
| 第6-8週 | テストステロン・肝機能・プロラクチン | テストステロン・血清カリウム・腎機能 |
| 3-6ヶ月ごと | 肝機能・プロラクチン・累積用量レビュー | 血清カリウム・腎機能 |
| 年1回 | 頭部MRI（長期ユーザー） | 乳腺スクリーニング |

## 中国本土での入手状況

- **CPA**：「色谱龙」（アンドロクール）50 mg規格が主流。一部の病院では国産「酢酸シプロテロン錠」が処方可能。精神科、内分泌科、生殖医療科で処方されることが多い。1箱あたり約30-80元、保険適用は限定的
- **スピロノラクトン**：「アルダクトン」20 mg/40 mgが主流、循環器内科、腎臓内科、皮膚科（ざ瘡適応）で取扱あり。安価で保険適用も広い

**闇市場／海外通販**：偽薬、保管温度の管理不全、追跡不能などの問題が深刻なため非推奨。

## 選択方法（臨床判断フレームワーク）

**CPAを優先する状況**：
- 血清T値を女性範囲まで下げたい
- スピロノラクトン単剤でT抑制が不十分
- 腎機能の問題または高カリウム傾向がある
- スピロノラクトンの利尿系副作用に耐えられない

**スピロノラクトンを優先する状況**：
- 長期HRT（5年超）を計画する若年ユーザー（累積用量に敏感）
- 髄膜腫の既往またはベースラインMRI異常
- 同時に降圧または心血管保護が必要
- CPAが入手困難または経済的負担が大きい

**GnRHアゴニストを検討する状況**：
- CPAとスピロノラクトンの両方に禁忌／不耐容
- 最も完全なT抑制が必要（性腺摘出術前の移行期）
- 経済的に余裕がある

**E2注射のみ／高用量単剤**：
- 一部のユーザーは高用量E2注射でTが自然抑制され、抗アンドロゲンを中止できる
- 必ず医師指導下で継続的な血液モニタリングを行うこと
