## レビューチームの構成

HRT薬典の医学コンテンツは、主題ごとに分担する分野横断のボランティアチームがレビューします。

| 領域 | レビュー責務 |
|---|---|
| 内分泌科 | エストロゲン／抗アンドロゲン／プロゲストーゲンの用量、血液検査の解釈、代謝と骨密度 |
| 精神科 | ジェンダー肯定評価、気分の変動、併用薬の相互作用 |
| 肝臓内科／血液内科 | CPA の肝毒性、VTE リスク、凝固モニタリング |
| 法務アドバイザー | 中国本土の処方コンプライアンス、プライバシー法、医薬品の国際持ち込み |

## レビュアー一覧

> ボランティアの方々のアイデンティティ表明を尊重します。実名公開に同意したアドバイザーは下記に氏名／ORCID／所属機関を記載し、非公開を希望する方は職種と所属診療科のみを記し、識別可能な情報は開示しません。

**現在のステータス（2026-04-19）**

当サイトは現在、**「コミュニティ整理＋国際ガイドラインのクロス引用」** の段階にあり、**実名公開の有資格医師との契約はまだ行われていません**。

- 医学的内容はすべて以下の権威ある出典に準拠し、DOI を付して引用しています。
  - Endocrine Society 2017《Endocrine Treatment of Gender-Dysphoric/Gender-Incongruent Persons》
  - WPATH SOC 8 (2022)
  - UCSF Transgender Care Guidelines
  - EMA／FDA の医薬品安全情報
- すべての医学的主張に `<CitationRef>` を付け、原文献を指し示します
- ページの frontmatter で `evidenceLevel`＋`lastReviewed`＋`references` の 3 つのメタデータを必須化しています

**次のステップ**：中国本土、香港・マカオ・台湾、日本、北米のトランスフレンドリーな有資格医師に対して、レビューローテーションへの参加をお声がけしています。ご関心をお持ちの方は [フィードバックフォーム](https://wj.qq.com/s2/16576000/5fe5/) または GitHub からご連絡ください。

## レビュー原則

- **処方関係なし**：レビュアーは当サイトが言及する医薬品／ブランドと直接の処方関係を持たず、利益相反を回避します
- **異論の記録**：レビュー意見と編集判断は全行程で記録し、重大な相違は公開的に開示します
- **アイデンティティ保護**：アドバイザーは匿名を選択できますが、編集部に対しては資格証明と所属機関証明の写しを提示する必要があります
- **更新義務**：ガイドラインの重大更新から 48 時間以内に、レビュアーが既存ページへの影響を確認します
