## 写给回访者

如果你记不清该看哪一篇，这里是快速判别：

- **打第一针前**，从 [首次注射完整指南](/zh/guides/first-injection/) 开始
- **已在用某种抗雄，指标异常或买不到**，看 [抗雄切换完整指南](/zh/guides/switch-antiandrogen/)
- **已在用某种 E2 途径，想改其他途径**，看 [切换 E2 给药途径完整指南](/zh/guides/switch-e2-route/)

更广的起点：

- 还没开始用药 → [用药前：你准备好了吗？](/zh/before-you-start/)
- 不清楚自己在哪个阶段 → [用药路径图](/zh/pathway/)
- 想对比多个药物的参数 → [药物比较器](/zh/tools/drug-comparator/)

## 后续规划

现有指南覆盖三个最常见决策场景。未来会按社区反馈补充其他任务型指引（如首次血检报告解读、HRT 与心理咨询的协调、长期随访节奏）。如果你在找的场景还没有，可以通过 [AI 问答助手](/zh/tools/ai-assistant/) 先获得初步信息，并在 [关于本站](/zh/about/) 留言反馈。
