## 内容纠错

发现医学内容有误？请通过 GitHub Issue 提交结构化纠错报告。

**需要提供：**
- 出错的页面 URL
- 当前错误的具体内容（引用原文）
- 建议修正为什么
- 支持修正的参考文献（DOI 或公开链接）

## 纠错处理流程

1. **收到** — 你提交的纠错进入审核队列
2. **评估** — 编辑部核实引用文献和建议修正的准确性
3. **修正** — 更新页面内容并标注更新日期
4. **回复** — 在 GitHub Issue 或原帖回复确认已修正

| 类型 | 预计响应时间 |
|------|-------------|
| 安全相关（剂量/禁忌/急症） | 48 小时内 |
| 医学数据错误 | 1 周内 |
| 措辞/翻译问题 | 1-2 周 |
| 功能建议 | 视排期 |
