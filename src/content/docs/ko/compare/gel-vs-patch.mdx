젤과 패치는 모두 경피 에스트라디올로, **간 초회통과 대사를 완전히 우회**하며 VTE 위험은 기준치에 근접합니다<CitationRef id="canonico-2018" />. 둘은 유럽·미국에서 40세 이상 HRT의 1차 선택이며, 중국어권에서는 입수가 다소 어려우나 주목할 가치가 있습니다.

:::note[30초 결정 매트릭스]
| 내 상황이 → | 추천 |
|---|---|
| 피부 민감 / 습진 병력 / 패치 점착제 알레르기 | **젤** · 접촉 면적이 작고 점착제 자극 없음 |
| 발한 많음 / 헬스 / 야외 작업 | **패치** · 4시간 샤워 금지 윈도우 없음 |
| 용량을 유연하게 미세 조정(펌프 수로 세부 조정) | **젤** · 1펌프 약 0.75 mg E2 |
| 매일 도포가 번거로움 / 출장 잦음 / 일정 불규칙 | **패치** · 주 1-2회, 혈중 농도 가장 안정 |
| 중국 본토 입수 용이(1선 도시 3차 종합병원) | **젤** · 에스트로젤 처방 가능, 패치는 해외직구 필요 |
| 영아/파트너와 밀접 접촉 시 transfer 우려 | **패치** · 부착 부위 전이 위험 극히 낮음 |
| 임신 / 수유 / 18세 미만 | 모두 권장 안 함, 진료받으세요 |

> 확신이 안 서면 자세한 비교를 계속 읽거나 [위험 자가평가](/ko/tools/risk-screener/)를 참조하세요.

## 한눈에 보기

| 항목 | 젤(Oestrogel) | 패치(Estraderm / Climara) |
|---|---|---|
| **투여 빈도** | 1일 1-2회 | 3-7일마다 1회 |
| **용량 조정** | 유연(펌프 수/누름 횟수) | 제한적(규격 고정, 절반 절단 가능) |
| **혈중 농도 안정성** | 일내 작은 변동 | 가장 안정 |
| **피부 반응** | 드묾, 국소 발적 | 10-20%가 부착 부위 가려움/홍반 |
| **Transfer 위험** | 도포 후 4-6시간 내 존재 | 극히 낮음(부착 부위) |
| **수영 / 샤워** | 4시간 후 가능 | 일부 패치 가능, 설명서 확인 필요 |
| **중국 본토 접근성** | 일부 3차 종합병원 처방 가능(에스트로젤) | 어려움, 대부분 해외직구 필요 |
| **월 비용** | 약 80-150위안 | 약 100-200위안 |

## 약동학

### 젤

피부에 도포하면 에스트라디올이 각질층을 확산하여 모세혈관에 진입합니다. 혈중 농도<CitationRef id="kuhl-2005" />：

- 도포 후 4-6시간에 상승 시작, 10-12시간에 피크
- 1일 1.5-3 mg(에스트로젤 1-2펌프) 시 정상 상태 50-150 pg/mL
- 일내 작은 피크-골, 연속 3-5일에 정상 상태 형성

### 패치

경피 패치는 폴리머 매트릭스로 방출 속도를 제어합니다<CitationRef id="hembree-2017" />：

- 첫 부착 6-12시간에 정상 상태
- 0.05 mg/일 규격 시 정상 상태 약 40-60 pg/mL
- 0.1 mg/일 규격 시 정상 상태 약 80-120 pg/mL
- 패치 교체 시 혈중 농도가 거의 변동하지 않음

## 사용 비교

### 젤

**권장 부위**：대퇴 안쪽 · 아랫배 · 팔 안쪽 · 윗등. 유방, 생식기, 얼굴 회피.

**조작**：
1. 깨끗하고 건조한 피부(발한 상태 아님)
2. 펌프 수만큼 짜내어 손바닥 또는 티슈로 고르게 도포
3. 5-10분 자연 건조 대기 후 옷 착용
4. 도포 후 손 씻기(전이 방지)

**금기 시간대**：
- 도포 후 4시간 이내 샤워, 수영, 사우나, 격렬한 발한 회피
- 도포 후 4-6시간 이내 타인 또는 반려동물 피부 직접 접촉 회피; 영아 밀접 접촉 시 옷으로 격리 필요

### 패치

**권장 부위**：둔부 상외측 · 아랫배 · 아랫등. 동일 부위 반복 회피를 위한 순환 사용.

**조작**：
1. 깨끗하고 건조한 피부(보습제 없음, 발한 상태 아님)
2. 보호 필름을 떼고 10-20초 누르기
3. 첫 24시간 격렬한 발한과 장시간 침수 회피

**자주 발생하는 문제**：
- **탈락**：격렬한 운동, 샤워 수온 과다 시 점착력이 떨어질 수 있음. 의료용 테이프로 보강 가능
- **피부 반응**：10-20% 사용자가 부착 부위 가려움 또는 홍반 경험. 매번 부위 변경으로 완화 가능; 지속 시 젤로의 전환 고려

## VTE와 대사 안전성

둘 다 대규모 연구에서 일관되게 다음을 보임<CitationRef id="canonico-2018" /><CitationRef id="coleman-2022" />：
- VTE 위험이 약물 미사용 기준치 수준
- 간 SHBG, 응고인자, CRP에 대한 영향이 매우 작음
- 담즙정체 위험 0에 근접

**이는 또한 40세 이상 MTF 또는 VTE 위험 요인이 있는 사용자의 국제적 1차 경로이기도 합니다.**

## 용량 선택

### 젤

| 브랜드 | 규격 | 일반 시작 | 일반 안정 |
|---|---|---|---|
| Oestrogel(에스트로젤) | 0.06% 젤, 1펌프 ≈ 0.75 mg E2 | 2펌프/일 | 3-4펌프/일(1-2회 분할) |
| Estrogel(미국판) | 0.06%, 1펌프 = 0.75 mg | 동일 | 동일 |
| Divigel | 포장 0.25/0.5/1 mg | 0.5 mg/일 | 0.75-1.25 mg/일 |

**용량 조정**：6-8주마다 혈액검사로 조정, 매번 ±1펌프 또는 반 펌프.

### 패치

| 브랜드 | 규격 | 빈도 |
|---|---|---|
| Estraderm MX | 0.025 / 0.05 / 0.075 / 0.1 mg/일 | 3-4일마다 |
| Climara | 0.025 / 0.0375 / 0.05 / 0.075 / 0.1 mg/일 | 7일마다 |
| Vivelle-Dot | 0.025 / 0.0375 / 0.05 / 0.075 / 0.1 mg/일 | 3-4일마다 |

**시작**：0.05-0.1 mg/일; **안정**：대부분 0.1 mg/일, 일부는 2장 중첩(0.15-0.2 mg/일) 필요.

**용량 유연성 낮음**：2장 중첩으로 중간 용량 모방 가능; 절반 절단은 매트릭스 방출 제어를 파괴하므로 **권장하지 않음**.

## 중국 본토 접근

- **젤 - 에스트로젤(Oestrogel)**：수입약, 1선 도시 3차 종합병원 산부인과/내분비과에 재고 있음. 가격은 1튜브당 약 120-180위안, 4-6주 사용 가능
- **패치**：중국 국내에 정규 채널이 거의 없음. **해외직구 경로**(합법성 자가 판단)：미국 Amazon, eBay, 홍콩 약국(처방 요건 다양)
- **보험 적용**：극소수 지역에서 부분 환급, 대부분 자비

## 선택 방법

**젤 우선**：
- 용량을 세밀하게 조정하고 싶음
- 패치 피부 알레르기 병력
- 매일 도포의 의식감을 수용 가능

**패치 우선**：
- 가장 안정적인 혈중 농도 추구
- 매일 도포 불편(출장, 일정 불규칙)
- 피부의 가끔의 발적·가려움을 개의치 않음

**둘 다 적합하지 않은 상황**：
- 중증 피부 질환(습진, 전신형 건선)
- 매우 높은 용량 필요(패치 등가 >0.2 mg/일) → 주사 고려
