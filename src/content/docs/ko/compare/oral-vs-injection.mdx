경구 에스트라디올(프로기노바/발레르산 에스트라디올 정)과 발레르산 에스트라디올 근주(EV)는 중국어권 MTF에서 가장 흔한 두 가지 E2 경로입니다. 본 페이지는 약동학, 안전성, 실무 진입장벽을 비교하며, **보편적 권고를 제시하지 않습니다**.

:::note[30초 결정 매트릭스]
| 내 상황이 → | 추천 |
|---|---|
| VTE 위험 최저화 / 연령 ≥40 / 흡연 / BMI>30 | **주사 EV** · 간 초회통과 우회, RR ≈ 1 |
| 주사 무서움 / 일정 불규칙 / 시술 학습 거부 | **경구 프로기노바** · 진입장벽 0 |
| VTE 병력 / 전조 동반 편두통 / Factor V Leiden | **주사 EV**(또는 경피) · 경구 금기 |
| 중국 본토 접근 용이 + 번거로움 회피 | **경구 프로기노바** · 산부인과 상비 |
| 단일제 치료 목표(항안드로겐 중단) | **주사 EV** · 약 82%가 T&lt;50 ng/dL 도달 |
| 주사 간격 불규칙 / 일내 안정성 추구 | **경구 BID** · 주사 피크/골 비 3-4:1 |
| 임신 / 수유 / 18세 미만 | 모두 권장 안 함, 진료받으세요 |

> 확신이 안 서면 자세한 비교를 계속 읽거나 [위험 자가평가](/ko/tools/risk-screener/)를 참조하세요.
:::

## 약동학

### 경구：간 초회통과가 곡선을 형성

경구 에스트라디올 또는 발레르산 에스트라디올은 소장에서 흡수된 후 문맥을 통해 간으로 들어가, **초회통과 대사로 50-70% 손실**됩니다<CitationRef id="kuhl-2005" />. 체순환에 도달하는 유효 용량은 복용량의 30-50%에 불과하며, 간은 고농도의 E2/E1에 "잠기게" 됩니다.

- 경구 2-4 mg, 혈중 농도 피크 약 50-150 pg/mL, 4-8시간 후 하강
- 반감기 약 12-14시간(E2 형태로), 단 E1/E2 비율이 높음
- 간에서 SHBG, 응고인자, CRP 생성 증가 → VTE / 담즙정체 위험 상승<CitationRef id="canonico-2018" />

### 주사：근육 내 서방형, 더 평탄한 곡선

발레르산 에스트라디올 근주 후, 유성 용제가 근육 내에 데포를 형성하고 에스테르 결합이 천천히 가수분해되어 E2를 방출합니다<CitationRef id="aly-2021" />：

- 4 mg EV 근주, 24-48시간 후 피크 약 300-600 pg/mL, 이후 지수적 감쇠
- 반감기 약 4-5일; 5-7일 후 100-150 pg/mL로 하강
- **간 초회통과를 완전히 우회**, 응고에 대한 영향은 생리적 에스트라디올에 근접

## VTE 위험 비교

대규모 MTF 코호트와 폐경 후 HRT 연구 기반<CitationRef id="canonico-2018" /><CitationRef id="coleman-2022" />：

| 경로 | 상대 VTE 위험(기준치 대비) |
|---|---|
| 경구 에스트라디올 | **2-4배** |
| 결합형 에스트로겐(CEE, 도태됨) | 4-6배 |
| 설하 투여 | 1.5-3배 가능성(근거 제한적) |
| 경피 젤 / 패치 | **1배(기준치)** |
| 근주 발레르산 에스트라디올 | **1배(기준치)** |

**절대 위험은 여전히 낮음**：젊고 건강한 MTF의 기저 VTE는 약 1-2/1000인년. 흡연 + 비만 + 연령 >40이 더해지면 절대 위험은 크게 상승합니다.

**가이드라인 입장**<CitationRef id="coleman-2022" /><CitationRef id="hembree-2017" />：
- 연령 ≥40세 + 위험 요인 1개 이상 → 비경구 우선
- VTE 병력 → 경구 회피, 경피/주사 + 항응고 평가 우선
- 수술 전 2-4주 → 경피로 전환 또는 일시 중단

## 효과 발현과 전환

### 경구 효과 발현

- 복용 후 2-4시간에 피크, 유방 발달, 피부 변화 등 조직 반응은 1-3개월 후 출현 시작
- 정상 상태 T 억제는 약 4-8주(항안드로겐 병용 시)

### 주사 효과 발현

- 첫 주사는 24-48시간 내 피크이지만 단회로는 정상 상태 도달 불가
- 2-3회 투여(약 10-20일) 후 정상 상태 근접
- 전환 이행기에는 중첩 권장：첫 주사 1-2주 전부터 경구 유지, 주사 후 점진적으로 경구 감량

### 상호 전환

**경구 → 주사**：
1. 등가 용량 계산：경구 2-4 mg/일 ≈ EV 4-8 mg 7일마다
2. 첫 주사 후 2-4일 경구 유지, 이후 점진적 감량
3. 주사 4-6주 후 골 E2 측정으로 정상 상태 확인

**주사 → 경구**：
1. 마지막 주사 중단, 5-7일 대기(혈중 하강)
2. 경구 시작, 초기 용량은 보통 정상 상태 주사 등가량의 1-1.5배(간 초회통과 보상)
3. 2-4주 후 혈액검사로 조정

## 실무 진입장벽

### 경구

- 진입장벽 0, 정시 복용만 하면 됨
- 주의：프로기노바는 발레르산 에스트라디올 2 mg/정으로 "에스트라디올 2 mg"과 등가, 환산 불필요
- 누락 복용：12시간 내 추가 복용, 12시간 초과 시 건너뛰기

### 자가 주사

- 유성 제제 근주：23G 1-1.5인치 바늘(약 25-38 mm), 약물 흡인은 21G
- 주사 부위：둔부 후외측(서혜부 후상외측 사분면), 대퇴 전외측, 삼각근(소량 시)
- 소독：70% 알코올, 사용 전 바이알 마개를 30초간 닦기, 주사 부위는 시계 방향으로 닦기
- 요령：기포 제거, 천천히 주입(10-30초), 발침 후 30초 압박

<WarningBox>
**첫 주사 강력 권장**：친구 동석 · 응급 전화 준비 · 30분간 알레르기/혈관미주신경 반응 관찰. 자세한 절차는 본 사이트의 [첫 주사 가이드](/ko/guides/first-injection/)를 참조.
</WarningBox>

### 피하 주사(SC) 대안

유성 발레르산 에스트라디올은 **피하 주사**(복부, 대퇴 전면) 가능하며, 통증이 가볍고 반감기가 약간 더 깁니다. 바늘은 25G × 5/8인치(16 mm). 근거 수준은 중등도이며, 일부 가이드라인이 수용<CitationRef id="aly-2021" />.

## 중국 본토 접근

| 경로 | 경구(프로기노바) | 주사(EV) |
|---|---|---|
| 산부인과 | 상비, 처방 용이 | 일부 병원 처방 안 함 |
| 내분비과 | 비교적 용이 | 의사 태도에 따라 다름 |
| 생식센터 | 매우 용이 | 배란 유도 프로토콜에 흔히 사용 |
| 정신과(트랜스 외래) | 처방 가능 | 처방 가능 |
| 비용 | 20 mg × 30정 약 40위안 | 5 mg/mL × 10앰플 약 60-100위안 |
| 보험 | 부분 환급 | 부분 환급 |

## 선택 방법(임상 의사결정 프레임워크)

**경구를 우선하는 상황**：
- 젊음, VTE 위험 요인 없음, 근주 학습 거부
- 주사 접근 곤란 또는 의사 처방 거부
- 단기 가교, 몇 개월 후 전환 예정
- "매일 복약" 형식 수용도 높음

**주사를 우선하는 상황**：
- 연령 ≥40 또는 VTE 위험 요인(흡연/비만/가족력)
- 경구 흡수 불안정(위장 수술 병력, IBD)
- 더 평탄한 혈중 농도 추구
- 자가 주사 학습 의향

**패치/젤을 우선하는 상황**(본 페이지에서는 비교 안 했지만 종종 최적)：
- VTE 병력 또는 고위험
- 바늘 완전 회피 희망
- 용량 미세 조정에 민감
