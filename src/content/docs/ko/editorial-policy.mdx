## 절대 게재하지 않는 것

HRT약전은 다음 항목을 결코 게시하지 않습니다.

<DangerBox>
- 개인 맞춤형 처방 권고("당신은 X mg 을 복용해야 한다" 등)
- 의약품 구매 링크, 판매 대리, 추천·수수료 QR 코드
- 응급 증상(DVT, 간 손상, 수막종 등)에 대한 인식을 약화시키는 표현
- DOI 또는 인용이 없는 의학적 주장
- 제3자 추적 스크립트, 이용자 건강 데이터 저장
- AI 면책 고지나 안전 경고를 우회하는 설정
</DangerBox>

## 언어와 어조

- **단정적 표현 지양**: "반드시 / 무조건 / 영원히" 대신 "권장합니다 / 일반적으로 / 대부분의 경우"를 사용합니다
- **독자를 주어로**: "환자는 A 를 선택해야 한다"가 아니라 "당신은 A 또는 B 를 선택할 수 있다"로 씁니다
- **불확실성 명시**: 근거 수준 B/C/X 의 결론에는 "데이터가 제한적이다" 또는 "전문가 의견" 표지를 붙입니다
- **낙인 회피**: 당사자 커뮤니티가 사용하는 용어를 존중하고, 낡거나 과도하게 임상적인 표현은 사용하지 않습니다

## 인용과 저작권

- 모든 의학적 주장은 `CitationRef` 컴포넌트를 통해 출처를 명시합니다(ID 는 `src/data/references.json` 의 항목과 매칭)
- 인용은 DOI 를 우선하며, DOI 가 없는 경우 학술지 출판 정보 + 안정 URL 을 함께 표기합니다
- 도표는 자체 제작이거나 Creative Commons / 퍼블릭 도메인 자료를 사용하며, 전재 시 출처를 반드시 표기합니다
- 본 사이트의 자체 콘텐츠는 CC BY-SA 4.0, 코드는 저장소의 LICENSE 를 따릅니다

## 오류 정정

독자는 [피드백 폼](https://wj.qq.com/s2/16576000/5fe5/) 또는 [GitHub Issue](https://github.com/cantascendia/yakuten/issues) 를 통해 제출할 수 있습니다.

| 유형 | 응답 시한 | 처리 방식 |
|---|---|---|
| 사실 관계 오류 | 72 시간 내 1 차 회신 | 페이지 상단 **정오표 안내** + 변경 이력 |
| 인용 편차 | 72 시간 | 인용 교체 후 정정 로그 공개 |
| 오래된 정보 | 7 일 | 점진적 업데이트 + `lastReviewed` 갱신 |
| 위험한 권고 | 24 시간 내 긴급 내림 | 평가 후 수정하거나 비공개 유지 |

중대한 정정은 다음 채널에도 동기화됩니다.

- 사이트 내 `/ko/appendix-references/` 개정 이력
- RSS / Telegram / 이메일 구독자

## 민원과 이의 제기

- **콘텐츠 민원**: [피드백 폼](https://wj.qq.com/s2/16576000/5fe5/)
- **법무 / 저작권**: GitHub Issue 에 `legal` 라벨을 붙이거나 저장소 메인테이너에게 연락
- **개인정보 관련**: 본 사이트는 식별 가능한 개인 정보를 수집하지 않습니다. 만약 유출이 발견되면 24 시간 내 긴급 처리합니다

## 이해 상충 공개

- **상업 후원 없음, 의약품 광고 없음, 구매 수수료 없음**
- 운영 비용은 자원봉사자 자비와 오픈소스 커뮤니티 기부로 충당합니다
- 편집자와 의학 자문위원은 이해 상충 공개서에 서명하며, 콘텐츠와 직접 관련된 처방 / 상업 관계가 있을 경우 해당 주제에서 회피합니다
- 외부 링크에는 어필리에이트 식별자를 포함하지 않습니다. 인용한 병원 / 핫라인은 커뮤니티가 장기적으로 검증한 자원만 게재합니다

## 재검토 주기

| 페이지 유형 | 기본 재검토 간격 |
|---|---|
| 응급 증상 인식, 용량 레드라인, 위험 경고 | **6 개월** |
| 의약품 상세, 가이드 | **12 개월** |
| 도구, 정책, 방법론 | **12 개월** |
| 블로그 글 | 게시일 기준 12 개월 후 1 회 점검 |

가이드라인의 중대한 변경(EMA/FDA 안전 정보, WPATH / Endocrine Society 신판)이 발생할 경우 **48 시간 내 점진적 업데이트**를 시행합니다.
